## 🔌 **API Integration**

All themes fetch dynamic data from backend endpoints:

### **Wedding Data Endpoint:**
```
GET /api/weddings/{wedding_id}
```

**Response Structure:**
```json
{
  "id": "wedding_id",
  "bride_name": "Bride Name",
  "groom_name": "Groom Name",
  "scheduled_date": "2025-10-15T19:30:00Z",
  "location": "Wedding Venue",
  "theme_settings": {
    "theme_id": "floral_garden",
    "custom_font": "Great Vibes",
    "primary_color": "#f43f5e",
    "secondary_color": "#a855f7",
    "pre_wedding_video": "https://youtube.com/watch?v=...",
    "cover_photos": [
      "https://cdn.url/photo1.jpg",
      "https://cdn.url/photo2.jpg"
    ],
    "studio_details": {
      "studio_id": "studio_123",
      "name": "Studio Name",
      "logo_url": "https://cdn.url/logo.jpg",
      "website": "https://studio.com",
      "email": "contact@studio.com",
      "phone": "+91 1234567890",
      "address": "Studio Address",
      "contact": "Contact Person"
    },
    "custom_messages": {
      "welcome_text": "Welcome to our wedding",
      "description": "Join us for our special day"
    }
  }
}
```

### **Data Usage:**
- `bride_name` → Couple name display
- `groom_name` → Couple name display
- `scheduled_date` → Date/time formatting
- `location` → Venue display
- `theme_settings.theme_id` → Theme selection
- `theme_settings.custom_font` → Typography
- `theme_settings.primary_color` → Color theming
- `theme_settings.secondary_color` → Color theming
- `theme_settings.pre_wedding_video` → Video section
- `theme_settings.cover_photos[]` → Gallery images
- `theme_settings.studio_details.*` → Studio section
- `theme_settings.custom_messages.*` → Welcome text

**✅ No hardcoded content** - Everything is dynamic from API
